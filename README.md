Iris Flower Classification
Project Overview

This project uses Machine Learning to classify iris flowers into different species based on their physical measurements.

The model analyzes flower measurements and predicts the flower type.

Dataset

The project uses the Iris dataset, which contains measurements of iris flowers.

Features used:

Sepal Length

Sepal Width

Petal Length

Petal Width

Target classes:

Setosa

Versicolor

Virginica

Machine Learning Model

The model used in this project is:

Logistic Regression

The dataset is split into:

Training data (80%)

Testing data (20%)

The model learns patterns from the training data and predicts the flower species.

How It Works

Load the Iris dataset

Split data into training and testing sets

Train a Logistic Regression model

Predict the species of a flower based on its measurements

Example output:

Flower measurements: [6.3 2.5 5.0 1.9]
Predicted flower: virginica
